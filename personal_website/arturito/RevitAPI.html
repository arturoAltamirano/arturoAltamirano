Foundations - GalloTools
Revit is a BIM software than enables engineers to model and design any sort of pre-construction schematics of buildings or associated infrastructure. There is an API that enables a developer to program additional tools to perform essentially any additional action within the software. I, along with another classmate from Tulane, was brought on for the summer to help improve these tools, known as GalloTools.
The Revit API
This software is similair to designing a web page or dynamic/static application like this one. There are visual elements like buttons and windows, and then backend files designed to manipulate data or user input. While employed at Gallo, I designed and developed nearly a dozen different files designed to make engineers lives easier with custom tools. Below are individual examples of some of my favorite tools.
Dockable Pane
The Dockable Pane is essentially an anchorable persistent pop-up window designed to display information from previous tasks. For example, an engineer may filter all pipes within a shematic by material and general application, the resulting pipes from this filtration are included in the pane, and a hyperlink to their cordinates will take you to the pipe's coordinates within the schematic.
