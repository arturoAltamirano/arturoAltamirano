Foundations
While working at the NASA Michoud Assembly Facility, I was asked to configure a iSCSI hard drive duplicator. This device was fascinating to me, and the time I spent messing with it was quite enjoyable.
Workstations
This device was used to clone hard drive images from a series of Dell B2600 workstations. When these devices (pictured below) would malfunction and recquire repair or refurbishment, the sensitive calibration data stored on their drives would need to be stored in a safe space.
Outcomes
I was provided with a network hub to move the data and a Buffalo NAS to store the image data. Pictured right is the final product, a mini network sitting on a file cabinet. The hub simply redirects the data to every other port, and I configured the NAS to generate a new file every time a transmission was initiated.
