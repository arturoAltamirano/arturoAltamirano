The Perceptron and Linear Regression
The perception algorithm is a linear learning algorithm used for binary classification. You can see and assignment done for this class which displays it below.
Regularization and Legendre Transformation
Regularization and the Legendre Transformation can be used to more accurately predict the true value of a function. Where an unregularized hypothesis may be overly reactionary to fluctuations in small datasets, regularization can be used to smoothen your predicted outcomes.
Support Vector Machines
Support Vector Machines are supervised models that can be used for regression analysis and, in this case, augmented linear classification.
